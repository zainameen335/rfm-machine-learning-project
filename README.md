# RFM Machine Learning Project
## Description
A Streamlit web application for RFM (Recency, Frequency, Monetary) analysis using machine learning to segment customers based on their purchasing behavior.
## Features
- Upload your customer CSV file and preview the data
- Perform RFM segmentation and clustering
- Interactive visualizations with Matplotlib and Seaborn
- Download results and insights
- Deployable on Streamlit Cloud for easy sharing
- ## Live Demo
[Click here to view the app on Streamlit Cloud](https://rfm-machine-learning-project-pktpxyh9fjpjed4jggbrmq.streamlit.app/)
## Tech Stack
- Python 3.13
- Pandas, Numpy, Matplotlib, Seaborn
- Scikit-learn for machine learning
- Joblib for model serialization
- Streamlit for web app deployment
